HTML


<!DOCTYPE html> <!-- 宣告HTML版本為5 -->
<html> <!-- 從此至</html>的內容都是html -->
  <head> <!-- 從此至</head>都是網頁的相關資訊 -->
    <meta charset="utf-8"
    <title>My First WebSite!</title>
  </head>

  <body> <!-- 從此至</body>的內容都會顯示於瀏覽器主視窗的內容 -->
    <h1 id="hello" class="heading-one">Hello World!</h1>
  </body>
</html>

HTML


<h1 id="hello" class="heading-one">Hello World!</h1>
  • h1: 標記名稱(tag name)
  • <h1>: 開始標記(start tag)
  • id、class: 屬性名(attribute name)
  • "hello"、"heading-one": 屬性值(attribute value)
  • Hello World!: 元素的內容(Element content)
  • </h1>: 結束標記(end tag)

CSS


<!DOCTYPE html>
<html>
  <head>
    ...
    <style>
      h1 { color: red; }
    </style>
  </head>

  <body>
    ...
  </body>
</html>

CSS 選取器(Selector)


h1 {
  color: red;
}
  • h1: 類型選取器(element selector)
  • color: 屬性(property name)
  • red: 值(property value)

更多CSS 選取器(Selector)


h1 {
  color: red;
}

#hello {
  font-size: 10em;
}

.heading-one {
  font-weight: bold;
}
  • h1: 類型選取器(element selector)
  • #hello: ID選取器(id selector)
  • .heading-one: Class選取氣(class selector)

    width: 40%;
  }
}
